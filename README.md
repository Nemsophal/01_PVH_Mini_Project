# 01_PVH_Mini_Project
Stock Management System Group01

Technologies:
- Language: Java (JDK 21)
- MVC Pattern
- Database: PostgreSQL 18.1
- Build Tool: Maven/IntelliJ Project
- Database Connectivity: JDBC

Key features:
- Pagination
- Write product into database
- Read product with id
- Update using product id (update name, update unit price, update qty, update all)
- Delete product using id
- Search product by name
- Save insert/update
- Unsave insert/update (view)
- Backup
- Restore
- Set rows

Prerequisites:
stock_db database
pg_dump
psql
